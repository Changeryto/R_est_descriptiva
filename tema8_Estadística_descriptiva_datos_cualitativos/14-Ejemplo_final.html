Un ejemplo final

Vamos a llevar a cabo un análisis completo de un ejemplo con lo que hemos aprendido en esta lección.

El objeto HairEyeColor que tiene R es una tabla de frecuencias absolutas de 3 variables cualitativas: color de cabello, color de ojos y sexo.

Vamos a extraer de esta tabla, una tabla bidimensional de frecuencias absolutas de las variables Hair y Eye, sin distinguir el sexo. Para ello se puede sumar las subtablas de frecuencias para hombres y mujeres, aplicando as.table() al resultado finales para transformarlo en tabla si no lo fuera.

Taducir

Es recomendable traducir los nombres de las variables de esta tabla y sus niveles. Esto con la función dimnames() que se puede usar sobre DF. El resultado de aplicar esta función a una tabla es una list cuyos componentes son los niveles de cada variable.
